## Установка

``` bash
git clone https://github.com/dskyworks/kotiko.git
cd kotiko
npm i
```

## Использование
``` bash
# Запуск сервера
npm start
# Сборка в режиме разработки
npm run dev
# Сборка в продакшн
npm run build
```
